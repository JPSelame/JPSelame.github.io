US and Georgia State Map with Bar Graph

This project visualizes the US and Georgia State Map along with a bar graph showcasing the number of food-insecure children in different counties.
Prerequisites

    A modern web browser (e.g., Google Chrome, Mozilla Firefox).
    An internet connection to fetch required libraries.

Project Structure

graphql

.
├── index.html                   # Main HTML file.
├── script.js                    # JavaScript for the US Map.
├── script_georgia.js            # JavaScript for the Georgia State Map.
├── script_bargraph.js           # JavaScript for the bar graph visualization.
├── styles.css                   # Styling for the visualization.
└── README.md                    # This file.

Instructions to Run the Code

    Ensure you have all the files in the same directory.
    Double-click on the index.html file to open it in a web browser.
    You should see the US Map and Georgia State Map displayed.
    The bar graph showcasing food-insecure children in different counties will appear below the maps. Bars will animate sequentially.
